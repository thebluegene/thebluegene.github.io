MEAN Stack Polling Application


Created a full-stack polling application using MongoDB, ExpressJS, AngularJS, and NodeJS. Any user can view polls and vote in them. Results are displayed using the ChartJS plugin. If a user creates an account, they can create polls, add options to existing polls, and access an account page where they can view/delete polls they created. Code is available upon request.

Simon Memory Game


Created a 'Simon' game using Javascript and jQuery. The user must follow a random pattern that increments by one move when the user matches the previous pattern. To win the user must correctly match a computer pattern of 20 moves. If the user is wrong, the pattern is played again; however, if 'strict mode' is activated, the game restarts.

Wikipedia Viewer


Made with Javascript/jQuery using Ajax and the Wiki API, this application clones the functionality of the Wikipedia search bar. It displays up to 10 items after the search is submitted, and also gives auto-complete suggestions as the user types. The user can be taken to a random Wikipedia page by clicking the 'Random' button.

Created a tic-tac-toe game using Javascript and jQuery. The user can make the futile decision of whether to go first or second, and then ultimately discover that the computer is programmed to never lose (within the rules of tic-tac-toe, of course).

TwitchTV User App
